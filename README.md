USD/INR Macro Analysis Dashboard (2010–2025)

# Project Objective
To analyze long-term depreciation of INR against USD and identify macroeconomic drivers and risk characteristics using an interactive Power BI dashboard.

# Key Questions Solved
1. How much has INR depreciated cumulatively since 2010?
2. What are the volatility regimes in USD/INR?
3. What were the major drawdown periods?
4. How does USD/INR correlate with:
5. Brent Crude Oil
6. US 10Y Yield
7. Macro indicators?

# Key Insights
-INR depreciated ~88% from 2010 to 2025.
-High volatility regimes aligned with global macro stress periods.
-Rolling correlation shows dynamic relationship with oil prices.
-Major drawdowns observed during global risk-off events.
